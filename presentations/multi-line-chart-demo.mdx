## Feature Specifications

### Usage Example
```jsx
<ChartSection 
  title="Your Chart Title"
  type="line"
  data={[
    {name: "Jan", series1: 100, series2: 120, series3: 90},
    {name: "Feb", series1: 150, series2: 140, series3: 110}
  ]}
  dataKeys={["series1", "series2", "series3"]}
  seriesNames={["Actual", "Target", "Previous"]}
  colorScheme="primary"
>
Chart description here
</ChartSection>
```

### Key Features
- **dataKeys**: Array of data property names to render as lines
- **seriesNames**: Optional custom names for legend (defaults to dataKeys)
- **Automatic Legend**: Shows when multiple series are present
- **Color Assignment**: Uses categorical colors for multiple series
- **Enhanced Tooltips**: Shows all series values on hover
